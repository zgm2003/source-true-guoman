# Feed Alignment Rules

Use this reference for long scope, multi-chapter work, dense dialogue, source-span alignment, or local continuity checks before drafting final continuous feed lines.

## Source-span alignment

Map each requested source span to an intended feed line range before drafting. Preserve source order and coverage; do not move later events, dialogue, or reveals before their source setup.

## Private line/span plan

Make a private line/span plan that records the source span, intended feed line range, main visible beat, dialogue anchor, and asset dependency. Keep it private unless the user asks for planning notes.

## Coverage ledger

Track each source beat as kept, split into adjacent continuous lines, or reserved for a user-directed trim decision. Do not reduce coverage by reducing line count.

## Local continuity check

Before finalizing each numbered line, compare it with the previous visible state: location, posture, held objects, injuries, interface state, outfit, and speaker focus. The next line must remain compatible unless the source or prior line supports the change.

## Dialogue alignment

Keep long dialogue in adjacent continuous lines in source order. Use the speaker as the primary subject; reaction inserts may appear only when they do not hide or replace the speaking beat.

## Do not import fixed-second timelines

Do not import fixed-second timelines, fixed-duration groups, 15-second slices, or time-based batches. Feed line ranges follow source spans and continuity, not preset durations.
